Construction workflow

One ordered workflow assigns the strongest available motif category while preserving the motif links and model files behind each record.

input

TF records

Human UniProt records associated with transcription and DNA binding; non-TF DNA-associated functions filtered out.

known

Database match

Direct motif matches from JASPAR, CisBP and HOCOMOCO.

transfer

Nearest neighbour

MMseqs2 transfer from annotated TFs, split into close and intermediate identity tiers.

Examples

Common entry points for using the web database.

Search by TF

Find motif records for a factor

Use a gene name, UniProt accession or protein name. The TF page links motif records, generated PWMs, regions and 3D model files when available.

PPARG TP53 A0A0S2Z4K5

Search by family/domain

Inspect a DNA-binding class

Search PFAM identifiers, domain names or family labels to recover groups of TF records and compare their motif categories.

Homeobox PF00105 zf-C2H2

Scan DNA

Test a sequence against generated PWMs

Paste DNA or FASTA, choose a motif set, and inspect ranked motif occurrences with TF links and annotation categories.

>p53_site_test TTTTTTGAACATGTCCCAACATGTTGTTTTTT

HeaderUniProt accession, gene/protein annotation, organism, sequence length and category summary.
MotifsKnown, nearest-neighbour and ModCRE-derived motif links, with generated PWM status and source.
RegionsDNA-binding regions from UniProt/domain annotation and model-supported ranges.
3D modelsIndexed PDB files, model source, template/model identifiers and linked motif records.
DownloadsMEME motif files and model files where present in the local index.

DNA scanning output

The scanner reports motif occurrences, not new motif annotations. Results are ranked by motif match statistics and linked back to the TF/motif records used for the scan.

ModCRE and ModCRELib

ModCRE derives DNA-binding preferences from protein-DNA structural models. In ModCRE DB, these records are kept distinct from Known and nearest-neighbour motif assignments.

When indexed, the ModCRE/ModCRELib-derived model collection provides the 3D model files, model identifiers and structure-derived PWM records shown on TF pages.

ModCRE server ModCRE paper

Source databases

Known motifs are integrated from public PWM resources. Sequence metadata comes from UniProt and domain/family information from PFAM/InterPro-style annotations.
